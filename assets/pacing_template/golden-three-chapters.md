# 黄金三章结构模板

> 面向番茄小说平台 | 免费章追读转化核心

## 定义

黄金三章指第1-3章，是读者决定是否继续阅读的**生死窗口**。番茄平台数据显示，前三章流失率高达70-85%，黄金三章的唯一目标是让读者点击"下一章"。

## 字数建议

| 章节 | 字数 | 说明 |
|------|------|------|
| 第1章 | 2000-2500字 | 偏短，降低阅读门槛 |
| 第2章 | 2500-3000字 | 正常节奏，展开核心设定 |
| 第3章 | 2500-3000字 | 完成第一个小高潮 |

> 番茄免费章每章不宜超过3000字，过长导致读者中途退出。

---

## 前200字抓人模板

读者在前200字内决定是否继续。以下是四种高效开头模板：

### 模板A：危机开局

```
【场景：主角正在做某件事】
【第1句：感官冲击——用听觉/视觉/触觉切入】
【第2-3句：危机降临，制造紧迫感】
【第4-5句：抛出悬念钩子】

示范：
凌晨三点十七分，林远被手机震醒。
屏幕上是一条陌生号码发来的短信——"你弟弟在城西废楼，还有一个小时。"
他翻身下床时，发现卧室门从外面被反锁了。
```

### 模板B：身份反差

```
【第1句：主角当前处境（越低越好）】
【第2句：意外事件打碎日常】
【第3句：暗示主角隐藏身份/能力】

示范：
苏晚在出租屋里啃着最后一包泡面时，门被踹开了。
三个黑衣男人冲进来，为首那个单膝跪下："小姐，老爷子请您回去继承家业。"
她看了一眼泡面，又看了一眼黑衣男人。"……这面我能吃完再走吗？"
```

### 模板C：结果前置

```
【第1句：直接交代一个震撼结果】
【第2-3句：回溯原因（制造好奇心）】
【第4句：暗示这只是开始】

示范：
陆沉杀了一个人。
准确地说，他只是把那个人推下了天台，但那人死了。
现在他站在天台边缘，看着楼下的警车灯闪成一片，心想：还有两分钟，他们就会到顶层。
```

### 模板D：日常碎裂

```
【第1-2句：极度日常的场景（建立代入感）】
【第3句：一个不对劲的细节】
【第4-5句：日常彻底崩塌】

示范：
陈默每天七点十五分准时坐上地铁，靠在第三节车厢第三根立柱旁边刷手机。
今天他习惯性地抬头，发现对面坐着一个和自己长得一模一样的人。
那人也在看他，然后笑了一下，用口型说了三个字："别回家。"
```

## 章节目标

### 第1章：钩子章

**核心任务**：建立代入感 + 抛出悬念

- ✅ 主角出场，让读者喜欢/同情/好奇
- ✅ 世界观/设定在动作中自然交代（不要旁白解说）
- ✅ 结尾必须有一个**未完成事件**
- ❌ 不做大段背景介绍
- ❌ 不出现超过3个角色
- ❌ 不铺垫，直接进入事件

**结构**：
```
0-200字：抓人开头
200-800字：建立代入感，交代主角处境
800-1500字：核心事件发生，打碎日常
1500-2000字：悬念升级，结尾留钩
```

### 第2章：入坑章

**核心任务**：深化悬念 + 展示核心卖点

- ✅ 第1章钩子的延伸（不要换赛道）
- ✅ 核心能力/系统/设定在行动中展现
- ✅ 引入第一个配角（最好是对手/盟友）
- ✅ 章末抛出更大悬念
- ❌ 不要让主角一帆风顺
- ❌ 不要一次性解释所有设定

**结构**：
```
0-500字：承接第1章钩子
500-1500字：核心卖点展示（打脸/逆袭/揭秘/升级）
1500-2500字：遭遇更大阻力
2500-3000字：新悬念出现，结尾留钩
```

### 第3章：定型章

**核心任务**：完成第一个小高潮 + 确立故事基调

- ✅ 第一个小高潮爆发（爽点释放）
- ✅ 让读者看到主角的潜力/决心
- ✅ 建立长期目标（主线牵引力）
- ✅ 章末暗示更大的冲突
- ❌ 不要在这里完结一个完整事件
- ❌ 不要让主角变得太强（留上升空间）

**结构**：
```
0-500字：矛盾升级
500-1500字：小高潮爆发
1500-2500字：爽感释放 + 主角收获
2500-3000字：新的危机/目标出现，留长期钩子
```

## 番茄免费章 vs 付费章区别

| 维度 | 免费章（前30章左右） | 付费章 |
|------|---------------------|--------|
| 节奏 | 极快，几乎无废话 | 可适当放缓，允许铺垫 |
| 爽点密度 | 每章至少1个 | 每2-3章1个 |
| 字数 | 2000-2500字/章 | 2500-3500字/章 |
| 信息密度 | 高，快速推进 | 中等，允许世界观展开 |
| 钩子强度 | 每章必须有强钩子 | 钩子可以是情绪型 |
| 设定交代 | 在动作中带出 | 可以有小段旁白 |
| 付费点设计 | 第30章左右必须有强烈"想知道后续"的欲望 | 不适用 |

### 付费点（入V章）设计要点

- 通常在第25-35章设置付费门槛
- 付费章前1-2章必须制造**最强悬念**
- 让读者已经投入感情/时间，不愿放弃
- 免费章结尾应是"最想知道答案"的时刻

## 不适用场景

1. **严肃文学/文艺小说**：黄金三章模式过于商业，会破坏文学性
2. **慢热型悬疑**：需要长线铺垫的硬核推理（如东野圭吾式）
3. **纯文学实验**：意识流、碎片化叙事等
4. **IP向改编**：影视化小说可以放宽开头节奏
5. **已有大IP续作**：粉丝基盘大，可适当慢热
6. **女频纯爱/慢生活**：部分女频类型（年代文/种田文）节奏天然慢

---

## 注意事项

1. **番茄算法加权**：完读率 > 收藏率 > 点击率，前200字决定完读率
2. **移动端阅读**：读者在地铁/被窝刷，开头必须3秒内抓住
3. **同类竞品**：开书前看10本同类型番茄热门书的前三章
4. **标题+简介+封面**：与前三章风格统一，不要"骗点击"
5. **章节标题**：番茄读者习惯扫章节标题，每章标题应有信息量或悬念
