# 节奏曲线模板

> 面向番茄小说平台 | 卷级节奏设计

## 定义

节奏曲线是整卷/整本书的情绪波动图谱。核心原则：**开头紧、中段松、结尾紧**。读者的情绪像过山车，不是直线冲刺。

### 五个阶段详解

| 阶段 | 章节范围 | 情绪强度 | 核心任务 |
|------|---------|---------|---------|
| **开篇引爆** | 1-5章 | 7-9 | 黄金三章抓人，建立核心矛盾 |
| **第一波高潮** | 5-10章 | 8-10 | 第一个大爽点，读者确认追读 |
| **中段铺垫** | 10-20章 | 4-6 | 世界观展开，伏笔埋设，人物关系 |
| **蓄力爆发** | 20-30章 | 6-10 | 矛盾激化，逐步推向高潮 |
| **卷末高潮** | 30-40章 | 9-10 | 全卷最高潮，最强爽点+最强悬念 |

## 付费点设计

### 免费→付费转换策略

**核心逻辑**：读者在免费章投入了时间和情感，付费点前必须制造"不付费会死"的冲动。

### 付费点位置

```
免费章（25-35章）─────────────────→ 付费章
                                    ↑
                              付费点（入V章）
```

### 付费点前5章设计

| 章节 | 任务 | 节奏 |
|------|------|------|
| 免费-5 | 新矛盾出现，读者好奇心上升 | 中速 |
| 免费-4 | 矛盾升级，主角陷入困境 | 加速 |
| 免费-3 | 出现转机，但不完全解决 | 快速 |
| 免费-2 | 最大悬念揭露，答案呼之欲出 | 极快 |
| 免费-1 | **最强钩子章**——即将揭晓时戛然而止 | 停在最高点 |
| 付费-1 | 延续悬念，给部分答案，抛出更大问题 | 保持高位 |

### 付费钩子设计要点

1. **不要在免费章给完整答案**：让读者带着"差一点就知道"的感觉付费
2. **付费章前2章必须是最强剧情**：读者付费时不能失望
3. **身份/真相揭露**：最有效的付费钩子
4. **情感绑定**：让读者关心角色命运
5. **承诺后续**：让读者相信付费后有更多爽点

### 付费钩子示范

```
【免费章末尾——最强钩子】
林远站在仓库门口，手里握着那份DNA报告。
报告上写着：他找了十年的人，就在他身后。
他缓缓转过身——
（章节结束）

【付费章开头——延续+释放】
站在他身后的是一个白发老人。
老人看着他，眼眶泛红："远儿，你终于找到我了。"
"你是……"
"我是你爷爷。你爷爷没有死。"
"那你为什么要假死十年？"
老人的笑容消失了。"因为十年前，有人要杀我们全家。"
"谁？"
"你父亲。"
```

## 节奏档位定义

### 慢节奏（Slow）

**适用场景**：世界观展开、人物塑造、情感铺垫、日常过渡

**段落特征**：
- 长句为主（15-25字/句）
- 大量环境描写和心理描写
- 对话节奏舒缓，有停顿和沉默
- 章节结尾可无强钩子
- 每段落信息密度低

**示范段落**：
```
苏晚坐在阳台上，看着远处的江面。夕阳把江水染成了橘红色，几艘货船慢悠悠地驶过。她点了一根烟，烟雾在晚风中散开。三年了，她一直在追那个案子。有时候她会想，如果那天她早到十分钟，是不是一切都会不一样。烟燃到了手指，她才回过神来。手机屏幕亮了，是一条未读消息。她没看。
```

### 中节奏（Medium）

**适用场景**：正常剧情推进、对话交锋、线索推进、关系发展

**段落特征**：
- 中等长度句子（10-20字/句）
- 动作和对话为主，穿插少量心理描写
- 对话节奏适中，有信息交换
- 章节结尾需要中等钩子
- 每段落信息密度中等

**示范段落**：
```
"你说那个案子，我查过了。"林远把一份文件推到桌上。
苏晚翻开文件，越看越皱眉。"这些人……都是同一天死的？"
"不是同一天。是同一个小时内。"林远说，"分布在三个城市，五个不同的地点。"
"怎么做到的？"
"这就是我想问你的。"林远靠在椅背上，"你是当年的经办人，你应该比我更清楚。"
苏晚合上文件，沉默了几秒。"你想让我重新查这个案子？"
"我想让你帮我。"
```

### 快节奏（Fast）

**适用场景**：战斗场景、追逐戏、揭秘时刻、高潮爆发、打脸反击

**段落特征**：
- 短句为主（5-12字/句）
- 大量动作描写，极少心理描写
- 对话简短有力，多为单句
- 章节结尾必须有强钩子
- 每段落信息密度极高

**示范段落**：
```
陆沉侧身躲过第一刀。反手夺刀。肘击。膝顶。三招之内，对手倒地。
"就这？"他看着地上的男人。
男人爬起来，掏出枪。
陆沉更快。
子弹擦过他的耳际。他抓住男人的持枪手，用力一拧。
"咔嚓。"骨折的声音。
枪掉了。男人惨叫。
"我再问一遍。"陆沉蹲下来，盯着他的眼睛，"谁派你来的？"
男人的眼神闪了一下。
就是这一下。
陆沉知道——他在看另一个人。
他猛地回头。
```

## 档位切换规则

### 切换节奏的信号

| 从 → 到 | 信号 |
|---------|------|
| 慢 → 中 | 新事件发生、新角色登场、冲突出现 |
| 中 → 快 | 战斗开始、真相揭露、矛盾爆发 |
| 快 → 中 | 高潮结束、开始收尾、人物反思 |
| 中 → 慢 | 过渡章、日常描写、世界观展开 |
| 快 → 慢 | 大事件结束、需要喘息（罕见，慎用） |

### 番茄平台节奏建议

- **免费章**：以中-快为主，慢节奏不超过20%
- **付费前期**：保持中-快，建立读者信任
- **付费中期**：允许慢节奏展开，但每5章至少1个爽点
- **卷末**：快节奏收尾，制造最强爽点

---

## 注意事项

1. **节奏不是均匀的**：同卷内也要有波动，不要一条线走到底
2. **慢不是拖**：慢节奏要有信息增量或情感推进，不是水字数
3. **快不是乱**：快节奏要读者看得懂，不要信息过载
4. **档位切换要自然**：不要突然从极慢跳到极快，需要过渡
5. **番茄特殊性**：免费章节奏整体要比其他平台更快
6. **章节末尾**：无论什么节奏档位，结尾都应有至少一个钩子
7. **男频vs女频**：男频整体偏快，女频可适当放慢
