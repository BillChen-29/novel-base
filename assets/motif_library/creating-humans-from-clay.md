# 母题：神用泥土造人 (Creating Humans from Clay)

## 一句话定义

神（通常是女娲或其他创世者）用泥土捏制或塑造出人类的形象，赋予其生命，创造了最初的人类。

## 叙事结构模板

1. **创世者的孤独/需要**：创世者感到孤独，或需要有人管理世界。
2. **获取材料**：创世者找到合适的泥土（通常是河边的黄土）。
3. **塑造过程**：创世者用手捏制泥人，精心塑造每一个细节。
4. **赋予生命**：创世者向泥人吹气、施法或用自己的精血赋予其生命。
5. **批量创造**：为了加快速度，创世者用绳子甩泥点，创造了更多的人——但这些人的质量不如手捏的。
6. **人的分化**：手捏的人成为贵族，甩泥点的人成为平民（社会分层的神话解释）。

## 承载的常见命题

- 创造者的匠心：每一个人都值得被精心塑造
- 速度与质量的矛盾：为了效率牺牲了精细度
- 社会分层的起源：不同的人有不同的"出身"
- 人与泥土的关系：人的身体来自泥土，最终也将回归泥土
- 创造的偶然性：甩泥点的意外创造了大量人类

## 可搭配的意象池

- 河边的黄土（创造人类的材料）
- 创世者的双手（精心塑造的工具）
- 绳子/柳条（甩泥点的工具）
- 吹气/精血（赋予生命的媒介）
- 泥人的形状（从粗糙到精细的过程）
- 烧制/晾干（使泥人定型的过程）

## 常见情境原型

- **女娲抟土造人**：最经典的泥土造人神话
- **批量创造**：女娲用绳子甩泥点，创造了大量的人
- **创造的分化**：手捏的人和甩泥点的人命运不同
- **不同泥土的造人**：用不同的泥土（金、银、铜、铁）创造不同的人

## 经典案例

- **《风俗通义》女娲造人**：女娲用黄土捏制人类
- **《淮南子》女娲造人**：更详细的造人过程描述
- **希伯来神话上帝造人**：上帝用尘土创造亚当（跨文化的相似母题）
- **希腊神话普罗米修斯造人**：用泥土和水塑造第一个人（跨文化的相似母题）

## 不适用场景

- 造人材料不是泥土（可以是木头、石头、植物等其他材料）
- 造人过程轻松简单（造人应该是一项需要耐心和技巧的工作）
- 没有赋予生命的步骤（泥土变成活人需要超自然的力量）
- 所有的人质量相同（泥土造人神话通常解释了人的差异）
