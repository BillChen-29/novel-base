# 母题：师徒反目 (Mentor-Betrayal)

## 一句话定义

导师与弟子因理念分歧走向对立，曾经的信任与敬仰转化为最深刻的背叛感。

## 叙事结构模板

1. **触发**：弟子发现导师的某个行为与自己信仰的理念相悖，或导师要求弟子做一件违背其原则的事。
2. **抵达**：弟子开始质疑——是自己看错了，还是导师一直如此？他回忆过去的教导，发现那些话语有了新的含义。
3. **冲突**：弟子试图与导师对质，但导师有自己的道理。双方的分歧不仅是方法论，更是价值观的根本对立。
4. **高潮**：背叛的真相完全揭开。导师可能为了更大的"善"做了恶，或弟子发现导师从一开始就在利用自己。
5. **收束**：弟子必须超越导师才能真正成长——但超越意味着否定曾经塑造自己的一切。他成为自己的导师，代价是永远的伤疤。

## 承载的常见命题

- 传承与超越：学生必须杀死精神上的父亲才能真正独立
- 理想主义的破灭：你信仰的导师原来也是普通人，甚至更糟
- 权力的腐蚀：导师的权威如何变成控制的工具
- 知识的代价：导师教给弟子的，弟子用来对付导师
- 父子关系的隐喻：师徒本质上是选择性的父子关系

## 可搭配的意象池

- 实验室/工作室（知识传承的物理空间）
- 师徒合影（过去关系的物证）
- 导师的著作/手稿（知识遗产的象征）
- 夜晚的办公室/实验室（秘密的场所）
- 讲台/黑板（权威的象征）
- 咖啡/茶（师徒交流的日常仪式）
- 门（关上=关系终结）
- 奖杯/证书（成就与代价）

## 常见情境原型

- **警界师徒**：老警察带新警察，新警察发现老警察与犯罪组织有牵连
- **医患师生**：医学导师的实验涉及伦理问题，学生面临举报与忠诚的抉择
- **商界师徒**：创业导师的商业手段越来越灰色，弟子必须选择站队
- **学术师徒**：导师的学术成果涉及造假，弟子发现后面临学术圈的压力
- **武术师徒**：传统师门中的理念冲突，新派与旧派的对决

## 经典案例

- **《星球大战》系列**：绝地武士的师徒关系与堕落，阿纳金与欧比旺的对决
- **《无间道》**：韩琛与卧底之间扭曲的"师徒"关系
- **《社交网络》**：扎克伯格与温克莱沃斯兄弟、与爱德华多的关系
- **《霸王别姬》**：程蝶衣与段小楼的师兄弟关系，艺术与现实的撕裂
- **《杀死比尔》**：师父比尔与"新娘"之间的血色师徒情

## 不适用场景

- 师徒之间没有真正的情感纽带时（反目缺乏痛感）
- 分歧仅是技术层面而非价值观层面时（缺乏深度）
- 弟子对导师没有真正的敬仰时（背叛不成立）
- 故事的核心是外部冲突而非关系冲突时（母题错位）
