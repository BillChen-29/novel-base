# 母题：被忽视的仙女——诅咒的起源 (The Neglected Fairy)

## 一句话定义

国王或王后举办宴会却没有邀请某位仙女/女先知，被忽视的仙女在宴会上降下诅咒：孩子将在某个特定时刻（如纺锤刺伤、十六岁生日）遭遇不幸。

## 叙事结构模板

1. **庆典**：国王为新生的王子/公主举办盛大的庆祝宴会。
2. **邀请**：所有仙女/女先知都被邀请，但有一位被遗漏（或被故意排斥）。
3. **诅咒**：被忽视的仙女不请自来，在宴会上宣布诅咒。
4. **缓解**：另一位善良的仙女将诅咒减轻（从"死亡"变为"沉睡百年"）。
5. **预言的应验**：在预言的时刻，不幸果然发生。
6. **破解**：只有特定的行为（如真爱之吻、完成某项任务）才能打破诅咒。

## 承载的常见命题

- 遗忘即冒犯：不被邀请比被敌视更令人愤怒
- 祝福与诅咒的对称性：在同一个庆典上，善与恶同时降临
- 家长的过度保护：试图避免诅咒反而促成了诅咒的实现
- 命运的不可逃避：越试图阻止，命运越是以意想不到的方式实现
- 爱与牺牲的救赎：只有真正的爱才能打破诅咒

## 可搭配的意象池

- 十二位仙女的宴会（祝福的仪式）
- 被遗忘的第十三位（被忽视的愤怒）
- 纺锤/纺车（被诅咒的工具）
- 沉睡的城堡（时间的凝固）
- 玫瑰荆棘（围绕城堡的障碍）
- 真爱之吻（打破诅咒的行为）
- 百年后的王子（跨越时间的拯救者）

## 常见情境原型

- **睡美人**：最经典的"被忽视的仙女"故事
- **小美人鱼**：未被邀请的仙女将祝福变为考验
- **白雪公主**：恶皇后的诅咒作为变体
- **希腊神话中的宴会遗漏**：厄里斯未被邀请导致特洛伊战争
- **珀尔修斯的出生**：阿克里西俄斯忽视神谕的警告

## 经典案例

- **格林童话第50号《睡美人》**：最典型的版本
- **佩罗《睡美人》**：法国文学版本
- **瓦格纳《尼伯龙根的指环》**：命运的诅咒
- **《哈利·波特》中的预言**：被忽视的预言的实现
- **《权力的游戏》中的诅咒**：被忽视者的复仇
- **《冰雪奇缘》中的魔咒**：诅咒与救赎

## 不适用场景

- 当故事不需要"被忽视者的愤怒"这一动力时
- 当叙事不需要命运的不可逃避性时
- 当主题不涉及祝福与诅咒的对称性时
