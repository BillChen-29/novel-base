# 母题：治水 (Controlling the Flood)

## 一句话定义

洪水泛滥成灾，文化英雄（如鲧、禹）及其协助者想方设法治理水患，包括堵塞、疏导、祛除水怪等方式。

## 叙事结构模板

1. **洪水降临**：洪水滔天，淹没大地，人类面临灭顶之灾。
2. **初次尝试**：英雄（如鲧）用堵塞的方式治水，但失败了——堵不如疏。
3. **方法转变**：新一代英雄（如禹）吸取教训，改用疏导的方式治水。
4. **艰难历程**：治水过程漫长而艰辛，英雄三过家门而不入，身体变形（化熊开山）。
5. **动物/神的帮助**：龙、神龟、千里马等动物协助治水，天神也给予帮助。
6. **成功治水**：水患终于被治理，大地恢复安宁，英雄建立了功业。

## 承载的常见命题

- 堵与疏的哲学：对抗自然不如顺应自然
- 代际传承：父亲的失败成为儿子的经验
- 牺牲与奉献：治水英雄放弃了个人生活，全身心投入公共事业
- 人与自然的搏斗：治水是人类与自然力量的正面交锋
- 集体协作：治水需要多方面的帮助，不是一个人能完成的

## 可搭配的意象池

- 洪水/滔天巨浪（灾难的具象）
- 息壤（能自己生长的神土）
- 禹步（治水时的独特步伐）
- 神龟/龙（治水的协助者）
- 三过家门而不入（治水者的牺牲）
- 九鼎（治水成功后的权力象征）

## 常见情境原型

- **鲧堵禹疏**：父亲用堵的方法失败，儿子改用疏导成功
- **禹化熊开山**：禹在治水过程中变形为熊，被妻子涂山氏看到后化为石头
- **神龟助禹**：神龟帮助禹疏通河道
- **应龙画地成河**：龙在前面开路，禹在后面疏通

## 经典案例

- **《山海经·海内经》鲧窃息壤**：鲧偷取息壤堵塞洪水，被天帝处死
- **《史记·夏本纪》禹治水**：禹疏通九河，三过家门而不入
- **《孟子》禹疏九河**：禹采用疏导的方法治理洪水
- **苗族治水神话**：少数民族版本的治水故事

## 不适用场景

- 治水过程轻松顺利（治水应该是一项艰巨的工程）
- 治水者没有付出个人代价（治水需要牺牲）
- 只有堵塞没有疏导（治水的核心智慧是疏导）
- 没有动物或神的帮助（治水往往需要多方协作）
