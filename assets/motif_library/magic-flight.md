# 母题：魔法逃亡 (Magic Flight)

## 一句话定义

英雄夺取恩赐后被守护者疯狂追击，不得不借助超自然的变形、障眼法和魔法障碍物一边逃跑一边拖延追踪者，完成惊心动魄且常带喜剧色彩的回归之旅。

## 叙事结构模板

1. **恩赐的盗取**：英雄通过智谋或力量从守护者手中夺取宝物
2. **追击开始**：守护者发现恩赐被夺，展开疯狂追击
3. **变形追逐**：英雄和追踪者不断变换形态——野兔变猎犬、鱼变水獭、鸟变老鹰
4. **抛下障碍**：英雄向后抛出物品化为山岳、海洋等障碍拖延追踪者
5. **逃脱成功**：英雄借助魔法、智谋或外力最终摆脱追击，携带恩赐回归

## 承载的常见命题

- 智慧胜于蛮力：面对绝对力量时，计谋是唯一的武器
- 变形的本质：万物皆流，没有什么形态是固定的
- 代价与牺牲：逃亡往往需要付出重大代价
- 生命力的不可遏制：无论被追赶得多紧，生命总会找到出路
- 两界的张力：恩赐的转移打破了旧有的宇宙平衡

## 可搭配的意象池

- 变形（野兔→鱼→鸟→麦粒）
- 向后抛出的障碍物（发刷变山、梳子变山、镜子变山）
- 追逐者与逃亡者的不断变换形态
- 被切碎的身体（拖延追踪者的代价）
- 燃烧的火焰/火山（追击的热度）
- 大海/河流（逃亡的通道）
- 大石头堵住通道（两界之间的最终阻隔）

## 常见情境原型

- **圭昂·巴赫的变形逃亡**：从野兔变鱼变鸟变麦粒，最终被吞入腹中获重生
- **伊邪那岐逃出黄泉**：抛下头饰变葡萄、梳子变竹笋、桃子击败追兵
- **珀耳修斯斩杀美杜莎后逃跑**：蛇发女怪追逐中不断扔出障碍
- **伊阿宋与美狄亚的逃亡**：肢解弟弟拖延追兵
- **俄罗斯兄妹的逃亡**：扔出发刷、梳子、镜子化为大山

## 经典案例

- **圭昂·巴赫与卡里德薇**：威尔士传奇中的变形追逐——从野兔到麦粒
- **伊邪那岐与伊邪那美**：日本神话中逃出黄泉的惊险追逐
- **伊阿宋盗取金羊毛**：美狄亚帮他用计逃脱，牺牲弟弟拖延追兵
- **伊尔库茨克萨满偷灵魂**：变成黄蜂蜇神使其松手，灵魂逃出瓶子
- **兄妹与水怪**：扔出发刷、梳子、镜子化为三座大山

## 不适用场景

- 英雄正面战胜追击者而非借助智谋逃脱
- 追逐缺乏真正的危险感和紧迫感
- 变形元素仅作为视觉奇观而无象征意义
- 逃亡过程中英雄未经历任何代价或牺牲
