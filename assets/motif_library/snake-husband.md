# 母题：蛇郎型（蛇精变夫）

## 一句话定义

父亲为蛇精所迫许嫁一女，唯幼女肯嫁蛇；蛇化为美男，姐姐嫉而杀妹夺位，妹妹死后变形复仇。

## 叙事结构模板

1. **触发**：父亲外出遇蛇精威胁，被迫许嫁一女。
2. **抉择**：诸女推诿，唯幼女（善良/天真/命运使然）答应嫁蛇。
3. **幸福**：幼女嫁蛇后反获幸福，蛇精实为俊美男子或蛇化美男。
4. **背叛**：姐姐嫉恨，杀害幼女，冒名顶替。
5. **变形复仇**：妹妹灵魂化为鸟→被杀→化为树/竹→被伐→最终变形物反噬姐姐，复仇成功。

## 承载的常见命题

- 善恶报应：善良者的苦难终有回报，恶人的贪欲必遭反噬
- 变形与身份：蛇/人/鸟/树——身份的流动与不可剥夺性
- 姐妹之争：血缘中的嫉妒与权力争夺
- 弱者的逆袭：最被轻视者反而获得最终胜利
- 命运的不可违抗：被安排的命运终会找到它的出口

## 可搭配的意象池

- 深山密林中的蛇洞/妖洞（危险与诱惑并存）
- 蛇蜕/蛇皮（身份转换的物证）
- 化为鸟/树/竹的遗物（灵魂不灭的象征）
- 姐姐的嫁衣/皇冠（篡位者的伪装）
- 幼女的纺锤/织机（勤劳善良的象征）
- 月夜下的蛇影（神秘与恐惧）

## 常见情境原型

- **现代豪门版**：父亲的公司被神秘收购方威胁，被迫安排女儿联姻，小女儿嫁入"蛇穴"般的豪门，遭姐姐算计
- **职场版**：姐妹同入一家公司，妹妹被分配到看似最差的岗位，却意外成功，姐姐嫉妒陷害
- **仙侠版**：蛇精报恩化为书生，与凡间女子结缘，遭妖界追杀

## 经典案例

- **《白蛇传》**：蛇精化女嫁许仙，虽为变体，共享"蛇与人婚"母题
- **格林童话《三片蛇叶》**：蛇的变形与报恩
- **日本民间故事《舌切雀》**：妹妹善良受报，姐姐贪婪受罚

## 不适用场景

- 缺乏超自然元素的纯现实主义叙事时
- 主角无明确的善恶对立关系时
- 不涉及身份变形或灵魂转世主题时
