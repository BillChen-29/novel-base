# 母题：狗耕田型（傻弟靠狗发财）

## 一句话定义

兄弟分家，弟只分到一条狗；弟用狗耕田获得意外财富；兄借狗耕田失败，杀狗；狗坟上长出树或竹，弟再次获利，兄效仿仍失败。

## 叙事结构模板

1. **分家**：兄弟分家，兄得全部家产，弟只得到一条狗（或先得小动物，辗转换成狗）。
2. **奇迹**：弟让狗拉犁耕田，狗竟犁出金银财宝（或田地大丰收）。
3. **嫉妒**：兄借狗去耕田，狗不肯配合（或耕出毒蛇/灾难），兄一怒杀狗。
4. **二次奇迹**：弟掩埋狗骨，坟上长出大树或翠竹，树/竹能实现愿望（或摇落下金银）。
5. **再模仿**：兄砍树/竹效仿，被树/竹砸死或一无所获。

## 承载的常见命题

- 善恶有报：善良的弟弟总有好报，贪婪的哥哥终遭惩罚
- 忠诚的力量：狗对善主的忠诚超越生死
- 嫉妒的毁灭性：模仿者的失败源于内心的不纯
- 底层的逆袭：最被轻视者最终获胜
- 自然的意志：自然（树/竹/狗）会自主选择站在谁的一边

## 可搭配的意象池

- 一人一牛一犁的田野（极简的贫穷之美）
- 狗在田间奋力拉犁的身影（忠诚与力量）
- 狗坟上破土而出的树/竹（死亡中的新生）
- 金光闪闪的犁沟（奇迹的视觉化）
- 兄长挥刀杀狗的画面（暴力与背叛）
- 树叶在风中沙沙作响如同低语（狗的亡灵）

## 常见情境原型

- **商业版**：兄弟合伙创业，哥哥背叛弟弟独占公司，弟弟白手起家反超，哥哥效仿再度失败
- **职场版**：老实人被欺负调到最差部门，却因祸得福发现新机遇
- **乡村版**：留守老人与老狗相依为命，老狗死后树上每年开出奇迹之花

## 经典案例

- **中国各地"狗耕田"故事**：广泛流传于汉族、苗族、土家族等
- **《灰姑娘》的变体**：善者被虐待后获得超自然帮助
- **格林童话《狗和麻雀》**：动物的忠诚与复仇

## 不适用场景

- 缺乏兄弟/姐妹之间的嫉妒与竞争时
- 没有"模仿—失败"的重复结构时
- 动物不扮演关键角色时
