# 母题：英雄之旅·单一神话 (Hero's Journey / Monomyth)

## 一句话定义

英雄从日常世界启程，穿越阈限进入超自然领域，经历考验获得恩赐，最终携智慧回归社群——这条"启程-启蒙-回归"的三阶段十七步道路是所有文化中共通的叙事原型。

## 叙事结构模板

1. **启程（分离）**：①历险的召唤——命运发出信号，日常世界出现裂缝；②拒绝召唤——英雄犹豫退缩；③超自然的援助——导师/守护者出现赠予法宝；④跨越第一个阈限——越过已知世界的边界；⑤鲸鱼之腹——被黑暗吞噬，象征性死亡。
2. **启蒙（考验）**：⑥考验之路——连续试炼净化灵魂；⑦遇到女神——神圣婚姻/阴阳合一；⑧妖妇的诱惑——面对欲望与罪疚；⑨与天父和解——超越对终极权威的恐惧；⑩奉若神明——超越二元对立的神性意识；⑪终极恩赐——获得生命的秘密。
3. **回归（归来）**：⑫拒绝回归——不愿离开极乐之境；⑬借助魔法逃脱——超自然追逐；⑭来自外界的解救——被世人拉回；⑮跨越归来的阈限——重返平凡世界；⑯两个世界的主宰——调和灵性与物质；⑰生活的自由——终极恩赐的实践。

## 承载的常见命题

- 成长与蜕变：人必须先"死"才能真正"生"
- 二元世界的统一：灵性与物质、内部与外部不可分割
- 自我的消解与重建：放下执着是获得智慧的前提
- 社群责任：个人启蒙必须回馈集体才有意义
- 普遍人性：所有文化的英雄故事遵循同一深层结构

## 可搭配的意象池

- 幽暗的森林（未知领域的入口）
- 门槛/边界（阈限的物理化）
- 怪物/恶龙（潜意识的阴影力量）
- 宝剑/护身符（精神武器与保护）
- 菩提树/世界之轴（宇宙中心）
- 迷宫（心灵探索的迷途）
- 金羊毛/圣杯（终极追求的象征）

## 常见情境原型

- **史诗英雄**：奥德修斯、埃涅阿斯、吉尔伽美什踏上漫漫归途
- **宗教创始者**：佛陀菩提树下成道后回转世间传法
- **童话冒险者**：王子穿越黑森林击败恶龙拯救公主
- **现代朝圣者**：普通人被迫面对人生危机，经历蜕变后重新理解生活
- **艺术家/创造者**：深入潜意识的黑暗后带回新的作品与洞见

## 经典案例

- **《星球大战》**：卢卡斯公开承认以坎贝尔的单一神话为蓝本
- **《西游记》**：唐僧师徒四人历经八十一难取经归来
- **佛陀成道故事**：从王宫出走→苦修→菩提树下降魔→传法
- **《奥德赛》**：奥德修斯十年漂泊最终回归伊萨卡
- **《黑客帝国》**：Neo从日常觉醒→进入矩阵→成为"The One"

## 不适用场景

- 纯日常叙事，缺乏超越性的精神维度
- 角色没有发生根本性的内在转变
- 故事不涉及"已知→未知→回归"的空间或心理移动
- 以解构或颠覆传统叙事为目的的后现代作品（需谨慎使用）
