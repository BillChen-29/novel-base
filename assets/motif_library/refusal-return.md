# 母题：拒绝回归 (Refusal of the Return)

## 一句话定义

英雄在获得启蒙的至福后不愿离开特殊世界，拒绝将恩赐带回人间——沉睡在极乐中不愿醒来，消融在永恒的狂喜中不愿分散自我——但生命和社会终将把他召唤回来。

## 叙事结构模板

1. **极乐的诱惑**：英雄在超自然世界中体验到深沉的平静与幸福
2. **不愿回归**：英雄质疑为何要回到充满苦难的尘世
3. **退隐的诱惑**：圣人消逝在狂喜中，英雄永远居住在极乐岛上
4. **社会的召唤**：世人来敲英雄的门，社会嫉妒超脱凡尘之人
5. **被迫回归**：英雄最终被带回——或自愿，或被外力拉回

## 承载的常见命题

- 启蒙与责任的张力：个人解脱与社群义务之间的矛盾
- 极乐的危险：沉溺于灵性体验可能成为另一种逃避
- 入世与出世：佛教的菩萨道——在涅槃前暂停，选择普度众生
- 社会的需要：社群需要英雄带回恩赐才能复兴
- 回归的代价：从永恒回到时间，从平静回到纷扰

## 可搭象的意象池

- 极乐岛/长生不老女神的居所（不愿离开的诱惑）
- 深沉的睡眠（逃避觉醒的退隐）
- 洞穴/山中的隐居处（与世隔绝的冥想）
- 注连绳（阻断退路的象征）
- 敲门声/召唤（社会的呼唤）
- 泥盆中的沉睡者（被困在至福中的灵魂）

## 常见情境原型

- **马赫昆达的永眠**：国王要求永无休止的睡眠作为最高恩惠
- **佛陀的犹豫**：成道后怀疑佛法是否能被传播
- **菩萨的暂停**：观世音在即将涅槃时选择留下普度众生
- **圣人的隐退**：修道者在山洞中达到完美后拒绝回到人间
- **天照大神的退隐**：太阳女神退回洞穴，世界陷入黑暗

## 经典案例

- **马赫昆达的长眠**：在大山子宫深处沉睡无数创世周期
- **天照大神退入洞穴**：世界陷入黑暗，众神用镜子和舞蹈引她出来
- **佛陀的成道后犹豫**：担心深奥的真理无法被凡人理解
- **观世音菩萨的誓愿**：在涅槃前发愿普度众生后再入涅槃
- **普罗米修斯的被缚**：盗火后被锁在岩石上——另一种形式的"不愿回归"

## 不适用场景

- 英雄始终愿意回归而无内在的挣扎
- 拒绝回归仅作为拖延情节而非深层的心理冲突
- 社会对英雄的需要未被充分呈现
- 故事未探讨"个人解脱"与"社会义务"的张力
