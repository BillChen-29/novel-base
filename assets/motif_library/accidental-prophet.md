# 母题：有言必中 (Accidental Prophet)

## 一句话定义
一个普通人因偶然说出的话恰好应验，被误认为先知，因此获得最大的幸福与最高的荣誉。

## 叙事结构模板

1. **偶然**：一个普通人（通常是穷人或书生）偶然说了一句话。
2. **应验**：这句话恰好应验了——他无意中说出了窃贼的名字，或预言了某件事。
3. **出名**：人们认为他有预言能力，纷纷来求他预言。
4. **继续应验**：他又偶然说中了几件事，名声越来越大。
5. **成功**：他因此当了官、发了财、或娶了公主。

## 承载的常见命题

- 偶然与命运：是命运在操纵，还是纯粹的巧合？
- 名声的虚幻：他的预言能力是假的，但名声是真的
- 社会的盲从：人们为什么相信一个偶然说中的话的人？
- 底层的逆袭：穷人因为一句话翻身，这是幸运还是讽刺？
- 预言的悖论：真正的预言者不会被相信，被相信的预言者往往是骗子

## 可搭配的意象池

- 深夜的自言自语（偶然预言的场景）
- 围观的人群（盲从的群众）
- 官府的大堂（预言被验证的场所）
- 公主的绣楼（预言带来的婚姻）
- 玉玺（偶然找到的宝物）

## 常见情境原型

- **偷鸡的预言**：丈夫发现妻子有情人，假装预言鸡的下落，找到了死鸡的证据
- **尿床的先知**：丈夫尿床，因此被认为能够预言
- **说出窃贼的名字**：穷人在街上自言自语，恰好说出了窃贼的名字
- **猜木箱里的东西**：穷人被要求猜木箱里有什么，他随口说出答案

## 经典案例

- **艾伯华类型190**：有言必中，全中国流传
- **民间"诸葛亮"故事**：被神化的智者形象
- **《三国演义》中的预言**：各种偶然应验的预言

## 不适用场景

- 主角是真正的先知时（缺乏偶然性）
- 故事重点在于预言的准确性而非社会讽刺时
- 没有"偶然应验"的喜剧效果时
