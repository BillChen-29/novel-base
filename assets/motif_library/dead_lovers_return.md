# 母题：亡灵骑马迎新娘 (The Dead Lover's Return)

## 一句话定义

死去的爱人骑着马回到人间，在月光下带走他的新娘，一路奔向墓穴的入口——"你害怕吗？""和你在一起，我不怕。"

## 叙事结构模板

1. **婚礼之夜**：新娘正在等待新郎来迎娶。
2. **骑马者出现**：一个骑马的黑衣人出现在门前，说他是新郎派来的使者。
3. **上马**：新娘上了马，马开始疯狂奔跑。
4. **真相揭露**：骑马者说出："月光皎洁，死者奔驰，你害怕吗？"新娘回答："和你在一起，我不怕。"
5. **墓穴**：马跑到一座坟墓前停下，骑马者是死去的新郎本人。

## 承载的常见命题

- 死亡不是终点：爱比死亡更强大
- 信任与恐惧：在最恐怖的情境中依然信任所爱之人
- 婚姻的永恒性：死亡也不能解除婚约
- 月光下的超自然：月亮是阴阳两界的通道
- 无法完成的结合：死者带回新娘，但无法真正与她在一起

## 可搭配的意象池

- 月光下的骑马者（死亡的浪漫化形象）
- 墓穴的入口（阴阳两界的门槛）
- 新娘的白色婚纱（纯洁与死亡的双重象征）
- 狂奔的马（命运的不可控制）
- 黑衣骑手（死亡的使者）
- 月光（照亮另一个世界的光）
- 坟墓上的裂缝（通往地下世界的入口）

## 常见情境原型

- **鬼魂迎亲**：死去的爱人回来完成未竟的婚约
- **月光下的问答**："你害怕吗？""和你在一起，我不怕。"
- **墓穴之门**：马跑到一座墓前停下
- **无法回头的旅程**：一旦上马就无法下来
- **新娘的忠诚**：即使知道真相也选择跟随

## 经典案例

- **俄罗斯民谣《骑手》**：最著名的版本
- **冰岛传说《诗人与亡灵》**：北欧变体
- **爱尔兰鬼魂传说**：凯尔特文化中的类似情节
- **哥特文学中的亡灵新郎**：从民间传说到文学创作
- **《聊斋志异》中的鬼妻**：中国变体
- **蒂姆·伯顿《僵尸新娘》**：现代电影改编

## 不适用场景

- 当故事不需要超自然元素时
- 当主题不涉及死亡与爱情的交织时
- 当叙事需要现实主义而非浪漫主义时
