# 母题：弃子英雄 (Abandoned Child Hero)

## 一句话定义
一个婴儿出生后被遗弃（因怪异外貌、不祥预兆或母亲的恐惧），但得到动物或自然力量的庇护，奇迹般存活下来，长大后成为拯救民族的文化英雄。

## 叙事结构模板

1. **神奇的出生**：婴儿的出生方式异于常人（踩巨人足迹受孕、吞食异物等）。
2. **怪异的外貌**：婴儿出生时是肉球/怪胎，被视为不祥之兆。
3. **多次遗弃**：母亲多次尝试遗弃婴儿（丢在小巷、树林、冰上），但每次都得到自然力量的保护。
4. **奇迹存活**：牛马避让、鸟翼覆盖、冰面不裂，婴儿最终被收回抚养。
5. **英雄的成长**：婴儿长大后发明农业/文化，成为民族的骄傲。

## 承载的常见命题

- 天命不可违：注定要成为英雄的人，即使被遗弃也无法被毁灭
- 自然的庇护：动物和自然力量会保护注定要成就大事的人
- 命运的反转：从被遗弃的弃儿到受人尊敬的英雄
- 文明的起源：农业等关键文化发明往往被归功于神异的英雄

## 可搭配的意象池

- 肉球/怪胎的婴儿形象
- 牛马小心翼翼避开的场景
- 鸟类用翅膀覆盖婴儿
- 冰面上的婴儿
- 种子在泥土中发芽（农业的隐喻）

## 常见情境原型

- **农业英雄**：后稷被弃三次，长大后发明农业
- **王者归来**：摩西被放在篮子中顺水漂流，后来成为民族领袖
- **神话的跨文化变体**：萨尔贡王、罗慕路斯等都被遗弃后成为英雄
- **被忽视的天才**：一个被社会排斥的人最终证明了自己的价值
- **命运的捉弄**：越是被试图消灭的力量，越是强大

## 经典案例

- **后稷三弃三收**：姜原踩巨人足迹受孕，生下肉球，三次遗弃均被保护
- **摩西**：被放在蒲草箱中顺水漂流，被法老女儿收养
- **萨尔贡王**：被母亲放在芦苇篮中抛入河中，被灌溉者收养
- **哪吒**：出生时是肉球，被父亲劈开
- **罗慕路斯**：被遗弃在台伯河畔，由母狼哺育长大

## 不适用场景

- 故事不涉及出生、遗弃或命运转折时
- 主题偏向平凡生活而非英雄传奇时
- 叙事风格偏向写实主义时
