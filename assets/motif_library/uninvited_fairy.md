# 母题：被忽视的仙女——宴会遗漏者的诅咒 (The Uninvited Fairy / The Omitted Guest)

## 一句话定义

国王为新生的王子举办盛大宴会，邀请了所有仙女/女先知，唯独遗漏了一位——她不请自来，在宴会上降下诅咒。

## 叙事结构模板

1. **庆典**：国王为新生的继承人举办盛大的庆祝宴会。
2. **邀请**：所有仙女/女先知都被邀请，但有一位被遗漏（或被故意排斥）。
3. **不请自来**：被忽视的仙女出现在宴会上，宣布诅咒。
4. **诅咒的内容**：孩子将在某个特定时刻（如纺锤刺伤手指、十六岁生日）遭遇不幸。
5. **缓解**：另一位善良的仙女将诅咒减轻（从"死亡"变为"沉睡百年"）。

## 承载的常见命题

- 遗忘即冒犯：不被邀请比被敌视更令人愤怒
- 祝福与诅咒的对称性：在同一个庆典上，善与恶同时降临
- 家长的过度保护：试图避免诅咒反而促成了诅咒的实现
- 命运的不可逃避：越试图阻止，命运越是以意想不到的方式实现

## 可搭配的意象池

- 宴会的华丽装饰（表面的欢乐）
- 空着的座位（被遗忘的存在）
- 被忽视者的愤怒（不被邀请的羞辱）
- 咒语的宣告（命运的宣判）
- 善良仙女的缓解（诅咒的减轻）
- 预言的实现（命运的兑现）

## 经典案例

- **《睡美人》**：最经典的"被忽视的仙女"故事
- **《小美人鱼》**：未被邀请的仙女将祝福变为考验
- **希腊神话中的金苹果**：厄里斯未被邀请导致特洛伊战争
- **《哈利·波特》中的预言**：被忽视的预言的实现
- **《权力的游戏》中的诅咒**：被忽视者的复仇

## 不适用场景

- 当故事不需要"被忽视者的愤怒"这一动力时
- 当叙事不需要命运的不可逃避性时
- 当主题不涉及祝福与诅咒的对称性时
