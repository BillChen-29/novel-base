# 母题：大地母亲 (Earth as Mother)

## 一句话定义
大地被拟人化为一位生育万物的母亲——她春天怀孕、夏天分娩、秋天收获、冬天休养。女娲是大地女神的化身，用泥土造人是大地母亲生育功能的神话表达。

## 叙事结构模板

1. **大地的拟人化**：初民将大地看作一个有生命、有性别、能生育的存在。
2. **万物由地生**：春天花木从大地"生"出，如同婴儿从母体降生。
3. **女娲的身份**：女娲既是氏族女始祖，又是大地之神——二者在原始思维中混融。
4. **泥土造人的逻辑**：大地女神用泥土造人，因为泥土就是她自己的身体。
5. **地母之祭**：社祭是中国最古老的宗教活动，祈求庄稼丰收和人丁兴旺。

## 承载的常见命题

- 母性崇拜：女性生殖力量是氏族存亡的关键
- 人与土地的血缘关系：人从土地中来，死后归于土地
- 农耕文明的根基：对于大地母亲的依恋是农耕民族的文化心理
- 社会组织的起源："社"既是祭祀大地的祭坛，也是国家的标志

## 可搭配的意象池

- 春天万物萌发的大地
- 子宫与生育的隐喻
- 社树与社石（生殖力的象征）
- 黄土与泥土（女娲造人的材料）
- 洞穴、水井等与女阴相似的自然形态

## 常见情境原型

- **游子思乡**：远走他乡的人带上一把故乡的泥土
- **丰收祭祀**：农耕社会对土地的感恩与敬畏
- **母系社会的回忆**：女神崇拜是母权时代的文化遗产
- **恋母情结**：人类对大地母亲永恒的依恋

## 经典案例

- **女娲以泥土造人**：黄土捏人、绳子甩泥点
- **珞巴族神话**：大地是一个产妇，生产时的痛苦翻滚就是地震
- **苗族神话**：地母左乳被刺伤，奶水干涸，种子不能发芽
- **河南淮阳人祖庙会**：至今保留着以黄土捏人、祭祀女娲的习俗
- **天穿节**：纪念女娲补天的节日，以煎饼象征补天

## 不适用场景

- 故事发生在非农耕文化背景时
- 主题不涉及土地、生育或母性时
- 叙事风格偏向都市/现代，缺乏乡土气息时
