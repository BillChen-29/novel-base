# 母题：妖窟 (Demon's Treasure Cave)

## 一句话定义

一对被赶出家门的穷夫妻在妖窟过夜，妖怪反而把珍宝交给了他们——因为善良的人才是珍宝的合法主人。

## 叙事结构模板

1. **触发**：一对善良的夫妻被恶毒的公婆赶出家门，身无分文，被迫在荒野中的一个妖窟里过夜。
2. **考验**：夜深时，洞中的妖怪出现，周围所有人都被纠缠、恐吓。但妖怪审视这对夫妻，发现他们心地纯洁。
3. **转折**：妖怪不但没有伤害他们，反而把自己看守的珍宝全部交给了他们，说这些珍宝本来就属于善良的人。
4. **收获**：夫妻带着珍宝离开，从此过上了富裕幸福的生活。
5. **讽刺收束**：曾经赶走他们的公婆后来变穷了，不得不来投靠他们。夫妻接待了他们。

## 承载的常见命题

- 善有善报：善良本身就是最大的财富通行证
- 因果报应：赶走善人的恶人终将自食其果
- 财富的道德性：珍宝会选择它的主人，不是你拥有它，而是它选择你
- 贫富翻转：社会地位的戏剧性逆转
- 妖怪的正义：超自然力量有时比人间更公正

## 可搭配的意象池

- 荒野中的破庙/山洞（危险与机遇并存）
- 深夜的火光（黑暗中的希望）
- 锈迹斑斑的箱子/宝匣（被忽视的财富）
- 恶婆婆的悔恨面容（因果的讽刺）
- 金银珠宝在火光中闪烁（意外的馈赠）

## 常见情境原型

- **落难夫妻**：被家族抛弃，却在绝境中获得意外财富
- **妖窟探宝**：胆大的人在危险之地发现宝藏
- **恶人回头**：曾经的施害者落魄后被曾经的受害者收留
- **穷人暴富**：一夜之间从赤贫到巨富的逆转

## 经典案例

- **艾伯华类型124**：广泛流传于全中国，浙江、江苏、广东、河南均有异文
- **袁枚《子不语》续编**：河南安阳版本
- 类似于西方"宝库历险"母题，但中国版本强调道德判断而非武力
- 与"灰姑娘"母题有结构相似：善良的底层人最终获得奖赏

## 不适用场景

- 故事中没有善恶对比时（缺乏道德张力）
- 财富的获得纯粹靠运气而非品行时（失去主题深度）
- 没有"忘恩负义"或"因果报应"的收束时（缺少讽刺感）
- 妖怪只是单纯的恶势力时（中国版妖怪有道德判断力）
