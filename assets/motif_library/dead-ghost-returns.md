# 母题：死鬼复生 (Dead Ghost Returns)

## 一句话定义
死鬼从坟墓中复活，追逐活人，或死去的父亲回来找儿子，带来危险，最终被经书、神像或天亮所制服。

## 叙事结构模板

1. **复活**：死人从坟墓中爬出，或通过雷雨复活。
2. **追逐**：死鬼开始追逐活人，通过"闻"（夺走呼吸）杀人。
3. **躲藏**：活人躲进庙宇、藏在关帝像后、爬上大树。
4. **搏斗**：死鬼用手抠进树里追赶，活人用经书击中死鬼。
5. **制服**：天亮鸡鸣，死鬼失去生命力，或被经书、神像制服。

## 承载的常见命题

- 死后的世界：死后并不安宁，死者会回来
- 信仰的力量：经书、神像、关帝可以驱邪
- 夜行者的恐惧：黑暗中的追逐是最原始的恐惧
- 死者的执念：死后仍有未了之事
- 光明的救赎：天亮是生的希望

## 可搭配的意象池

- 坟墓的裂缝（死鬼复生的出口）
- 经书碎片（信仰的武器）
- 关帝像（庇护所）
- 大树上的躲避（求生本能）
- 鸡鸣破晓（生的希望）
- 闻（夺走呼吸）的死亡方式

## 常见情境原型

- **棺材复活**：血滴在棺材上，尸体复活追逐活人
- **亡父寻子**：死去的父亲变成鬼回来找儿子
- **雷雨复活**：几个死鬼通过雷雨复活
- **梳头鬼**：女鬼在夜里梳头摘下脑袋，滚动的头颅追赶活人

## 经典案例

- **艾伯华类型113**：死鬼被杀
- **艾伯华类型114**：死鬼追踪
- **袁枚《子不语》**：大量死鬼复活的故事

## 不适用场景

- 死鬼只是背景而非核心角色时
- 故事重点在于超自然力量的展示而非恐惧体验时
- 没有"最后得救"的张力时
