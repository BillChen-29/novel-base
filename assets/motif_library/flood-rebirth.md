# 母题：洪水再造 (Flood Rebirth)

## 一句话定义

世界被大洪水毁灭后，少数幸存者（神的选民）在神的保护下躲过灾难，通过血亲婚配重新繁衍人类，完成了世界的再造。

## 叙事结构模板

1. **神的选民**：在洪水到来之前，神预先选择了少数人（通常是一男一女或一家人），将他们保护下来。选民因服从神旨或有恩于神而被选中。
2. **躲过洪水**：选民在葫芦、木箱、高山等处躲过滔天洪水，全世界只剩下了他们。
3. **血亲婚配的困境**：洪水后只剩兄妹（或母子、父女），要繁衍人类就必须乱伦，而乱伦禁忌又是极其严厉的。
4. **神谕与占卜**：通过占卜神意（滚石磨合、烟合、竹接等），证明天意允许兄妹婚配。
5. **人类再造**：兄妹结婚后生下怪胎（肉瘤、葫芦等），将其剁碎撒向四方变成人，或直接繁衍出后代，人类重新遍布大地。

## 承载的常见命题

- 灾难与重生：毁灭是新生的前提，世界在循环中不断更新
- 神的拣选：在灾难面前，神会保护那些善良或有功的人
- 乱伦禁忌：血亲不得婚配是人类文明的重要标志
- 婚姻制度的起源：洪水再造神话解释了人类婚姻制度的由来
- 生命的顽强：即使世界毁灭，人类仍能重新繁衍

## 可搭配的意象池

- 葫芦：洪水中的救命容器，兄妹在葫芦中躲过洪水
- 雷公的牙齿：雷公赠给选民的牙齿，种下后长出葫芦
- 石磨：兄妹各持一扇从东西山滚下，合在一起证明天意
- 竹子：被砍断后又能接活，证明天意允许婚配
- 肉瘤：兄妹婚后生下的怪胎，剁碎后变成人
- 洪水后的荒芜大地：人烟绝迹的世界
- 九尾白狐狸：祥瑞的征兆

## 常见情境原型

- **瑶族《伏羲兄妹》**：雷公赠牙齿→种出葫芦→兄妹在葫芦中躲洪水→滚石磨合→结婚→生肉瘤→剁碎变人
- **仡佬族神话**：人皇制了四批人，前三批都灭绝了，第四批是兄妹二人，心肠好被神仙选中
- **彝族《查姆》**：神造了三代人——独眼人、直眼人、横眼人，前两代被洪水淘汰
- **女娲兄妹婚**：唐代《独异志》记载女娲兄妹在昆仑山，以烟合为兆结为夫妻

## 经典案例

- **兄妹婚的普遍性**：汉族、苗族、瑶族、侗族、彝族等三十多个民族都有此类神话
- **乱伦禁忌的反映**：神话中对血亲婚配的种种费难与开脱，反映了人类对乱伦禁忌的认识
- **洪水后的世界再造**：从苏美尔到中国，洪水再造是世界各民族共同的母题
- **怪胎的象征意义**：生下肉瘤、葫芦等怪胎，象征着新的人类与旧人类不同

## 不适用场景

- 没有世界性灾难时（此母题需要毁灭性的洪水或其他灾变）
- 没有少数幸存者时（"神的选民"是此母题的核心）
- 幸存者之间没有血缘关系时（血亲婚配的困境是此母题的关键张力）
- 人类不是通过有性生殖重新繁衍时（此母题强调的是有性生殖的再造）
