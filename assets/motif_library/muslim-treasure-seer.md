# 母题：回回采宝 (Muslim's Treasure Recognition)

## 一句话定义

一个回回（西域商人）看见一件不起眼的东西，认出这是宝贝，想高价买下；所有者不知其价值，得到钱后试图自己使用宝贝，结果失败了——因为只有识宝者才知道如何使用。

## 叙事结构模板

1. **触发**：一个来自西域的回回（或南蛮子、江西人）来到一个村庄，看见一件普通的东西——可能是一只死蜈蚣、一根葱、一块石头。
2. **识宝**：回回认出这是稀世珍宝，出高价要买。所有者好奇，问他这东西有什么用。
3. **泄露**：回回讲了宝贝的用途，但没有讲全——或者故意保留了关键的使用方法。
4. **贪心**：所有者见钱眼开，用高价卖给了回回，但又偷偷留了一部分宝贝，或者回回走后自己试图使用。
5. **失败**：所有者使用宝贝时失败了——要么方法不对，要么时机不对，要么宝贝在错误的人手里失去了效力。回回拿走了真正的宝贝，或者宝贝被毁了。

## 承载的常见命题

- 识宝者的智慧：有些知识只有特定的人才拥有
- 贪婪的代价：不完全理解就贪图拥有，必然失败
- 东西方知识差异：回回代表着另一种知识体系
- 宝贝的道德性：宝贝选择识宝者，不是出价最高者
- 泄露天机的危险：知识一旦说出就无法收回

## 可搭配的意象池

- 回回深邃的眼睛（识宝者的异域智慧）
- 普通的农具/日常物品（被忽视的宝贝）
- 宝贝在月光下发光（宝贝的真实面目）
- 所有者懊悔的表情（贪婪的代价）
- 回回骑马远去的背影（知识的不可追回）

## 常见情境原型

- **识宝者与守财奴**：智慧与贪婪的对决
- **普通中的非凡**：最不起眼的东西可能是最珍贵的
- **知识的垄断**：有些知识无法传授，只能意会
- **西域商人**：回回作为文化他者的神秘形象

## 经典案例

- **艾伯华类型169**：广泛流传于浙江绍兴
- 金华地区也有大量异文
- 与"芝麻开门"（类型170）有紧密联系
- 体现了中国民间对西域/伊斯兰文化的想象

## 不适用场景

- 没有知识差异时（核心冲突是识与不识）
- 所有者没有贪心时（失去叙事动力）
- 没有异域文化背景时（回回是关键角色）
