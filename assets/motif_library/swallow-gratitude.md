# 母题：燕子报恩型（鸟的馈赠）

## 一句话定义

一人施恩于受伤的小鸟（燕子），小鸟康复后报以金银财宝；另一人模仿其行为，却因动机不纯而失败，反遭灾祸。

## 叙事结构模板

1. **施恩**：主角在路边或家中发现一只受伤的小鸟，悉心照料至康复。
2. **报恩**：小鸟飞走后衔来金银（或引领他到藏宝之处），主角因此致富。
3. **模仿**：另一人（贪婪的邻居/亲属）得知此事，故意伤害一只鸟，然后假装救治。
4. **失败**：小鸟（或鸟群）识破其伪善，不仅不报恩，反而报复——啄食其财物或使其遭受损失。
5. **教训**：善有善报，恶有恶报，真心与假意天壤之别。

## 承载的常见命题

- 真心与假意：同样的行为，发心不同则结果天壤之别
- 万物有灵：鸟类能感知人类的善恶
- 恩情的不可交易：善行不是投资，回报来自真心而非算计
- 模仿的破产：形式可以复制，灵魂无法复制
- 自然的奖惩法则：自然对真诚者慷慨，对虚伪者残忍

## 可搭配的意象池

- 血迹斑斑的羽毛（受伤的脆弱）
- 温暖的掌心托着小鸟（善意的具象化）
- 小鸟衔着金光飞来（报恩的浪漫）
- 故意折断鸟翅的黑手（伪善的暴力）
- 满地散落的金币/被啄毁的财物（报复的痛快）
- 春天归来的燕子（循环与希望）

## 常见情境原型

- **商业版**：创业者真心帮助陌生人，陌生人成为他的贵人；竞争对手抄袭他的"策略"却遭遇惨败
- **乡村版**：善良的老农救了一只受伤的鹰，鹰叼来鱼虾报恩；贪婪的邻居杀鹰取羽，遭鹰群围攻
- **都市版**：白领在雨天帮助流浪猫，猫的主人恰好是公司高管，暗中帮助她升职；同事故意接近流浪猫讨好高管，反被识破

## 经典案例

- **伊索寓言《农夫与蛇》的反面**：善待动物得到回报
- **中国各地"鸟报恩"故事**：燕子、鹤、乌鸦等多种鸟类
- **格林童话《蜂王》**：帮助昆虫获得回报的故事

## 不适用场景

- 缺乏"施恩—报恩—模仿—失败"的四段结构时
- 不涉及动物/鸟类的角色时
- 人为善行与伪善的对比不是故事核心时
