# 母题：错误的孩子——被掉包的谋杀 (The Wrong Child Murdered)

## 一句话定义

当凶手要杀害一群孩子时，聪明的兄弟姐妹调换了他们的位置/睡帽/衣服，结果凶手在黑暗中杀死了自己的孩子。

## 叙事结构模板

1. **威胁**：一个恶人（食人魔、继母、女巫）要杀害一群孩子。
2. **掉包**：最聪明的孩子发现了威胁，悄悄调换了孩子们的位置、睡帽或其他标记。
3. **黑暗中的谋杀**：恶人在黑暗中摸索，杀死了自己标记的那个孩子。
4. **真相揭露**：天亮后，恶人发现死的是自己的孩子而非受害者。
5. **变形或惩罚**：恶人因悲痛或罪疚变成鸟/兽，或受到其他惩罚。

## 承载的常见命题

- 以彼之道还施彼身：恶人的计划反而害了自己
- 智慧胜过暴力：最弱小的孩子用智慧战胜了最强大的敌人
- 黑暗中的盲目：在黑暗中，身份的标记变得至关重要
- 标记与身份：衣服、帽子、位置成为身份的替代物
- 悲痛的变形：极度的悲痛或罪疚导致人格的彻底改变

## 可搭配的意象池

- 黑暗中的房间（无知与危险的空间）
- 被调换的睡帽/衣服（身份的标记物）
- 熟睡的孩子们（脆弱的受害者）
- 恶人的摸索（盲目的暴力）
- 天亮后的发现（真相的揭露）
- 变形为鸟（悲痛的永恒化）

## 常见情境原型

- **阿伊冬杀死伊提卢斯**：调换睡帽后误杀自己的孩子
- **《格林童话》中的食人魔**：聪明的孩子调换了帽子
- **《一千零一夜》中的掉包**：兄弟姐妹的保护
- **忒弥斯托与伊诺**：调换衣服后的误杀
- **《聊斋志异》中的类似情节**：中国版本

## 经典案例

- **荷马《奥德赛》XIX**：佩涅洛佩讲述的夜莺故事
- **格林童话第43号《没有手的女孩》**：类似的掉包情节
- **埃斯库罗斯《阿伽门农》**：家族诅咒的延续
- **索福克勒斯相关残篇**：忒弥斯托的误杀
- **《权力的游戏》中的弑亲情节**：现代变体
- **《冰与火之歌》中的孩子掉包**：身份的混淆

## 不适用场景

- 当故事不需要"掉包"这一核心机制时
- 当叙事不需要黑暗中的身份混淆时
- 当主题不涉及智慧胜过暴力时
