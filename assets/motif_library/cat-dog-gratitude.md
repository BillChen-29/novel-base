# 母题：猫狗报恩型（动物忠诚）

## 一句话定义

一人养了猫和狗，家中宝物被窃；猫狗主动（或被骂后）去为主人偷回失物；主人对猫狗的忠诚从此深信不疑。

## 叙事结构模板

1. **养宠**：主角同时养了一只猫和一条狗，感情深厚。
2. **失窃**：主角因某种缘故得到一件宝物（金子/灵芝/神奇之物），却被贼人偷走。
3. **追讨**：猫狗发现宝物丢失，主动（或被主人责骂后）出发追讨。
4. **智取**：猫狗配合行动——猫潜入贼家偷回宝物，狗在门外把风/接应。过程中遇到各种险阻。
5. **团圆**：猫狗成功带回宝物，主人感激不尽，人与动物的情谊更加深厚。

## 承载的常见命题

- 动物的忠诚：超越言语的深情，猫狗对主人的爱不需要回报
- 失而复得：宝物的失去与寻回象征着人世间的一切终会回到该在的地方
- 团结的力量：猫与狗本性不同，但为了共同的主人可以合作
- 无声的爱：动物不会说"我爱你"，但它们用行动证明
- 贼与义的对立：窃贼的不义之财终被正义追回

## 可搭配的意象池

- 月光下猫的金色眼睛与狗的坚定脚步（动物的灵性）
- 贼人家中堆积的赃物（贪婪的具象化）
- 猫从窗户潜入的无声身影（潜行的优雅）
- 狗在门外焦急等待的低吠（忠诚的焦急）
- 宝物在月光下闪闪发光（失而复得的喜悦）
- 主人抱着猫狗痛哭的画面（感动的瞬间）

## 常见情境原型

- **家庭版**：老人的积蓄被骗子骗走，家中的猫狗跟着骗子找到了他的住处
- **古装版**：将军的传国玉玺被敌国细作偷走，将军的战马和猎鹰一路追踪
- **现代版**：主人的电脑被偷，宠物狗跟踪气味找到了小偷的藏身处

## 经典案例

- **中国各地猫狗报恩故事**：流传广泛的民间故事类型
- **格林童话《猫和老鼠合伙》**：动物之间的信任与背叛（反面）
- **日本《忠犬八公》**：动物忠诚的极致表达

## 不适用场景

- 缺少"失去—追讨—寻回"的叙事弧线时
- 猫狗不扮演关键角色时
- 故事不涉及动物的灵性或超自然能力时
