# 场景：高潮（Climax）

## 一句话定义
剧情矛盾集中爆发的场景，主角面对最大挑战，情感和动作达到顶点。

## 关联母题（发生什么）
- hero-vs-water-monster（英雄大战水怪）
- magic-duel（魔法对决）
- twelve-labors（十二试炼）
- sacrificial-king（牺牲之王）

## 关联角色（谁出场）
- trash-turning-genius-mc（废柴逆袭主角）— 逆袭时刻
- reborn-avenger（重生复仇者）— 复仇高潮
- moral-grey-survivor（道德灰色求生者）— 道德抉择高潮

## 关联技法（怎么写）
- rhythm-variation.md（节奏变化）— 加速节奏，短句爆发
- satisfaction-design.md（爽点设计）— 爽点集中释放
- sensory-detail.md（感官细节）— 增强沉浸感
- scene-description.md（场景描写）— 环境烘托气氛

## 关联风格（什么语感）
- 天渊风格："众人震惊"式烘托，腹黑幽默
- 游戏入侵风格：数据化叙事，精确展示力量对比
- 末日降临风格：快节奏短句，智谋型主角

## 关联节奏
- satisfaction-loop.md（爽点循环）
- pacing-curve.md（节奏曲线）

## 写作要点
1. 高潮前必须有充分铺垫（至少3章蓄力）
2. 节奏加速：句子变短，段落变碎
3. 爽点释放要具体：不要"他变强了"，要"他一拳打碎了对面的护体真气"
4. 高潮后必须有缓冲（慢档或中档）
