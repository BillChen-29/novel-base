# 场景：世界观展示（World Building）

## 一句话定义
通过角色行动和环境细节自然展示世界观，避免大段解释性叙述。

## 关联母题（世界观元素）
- three-tiered-universe（三界宇宙）
- cosmic-egg（宇宙之卵）
- body-of-god-becomes-world（神体化为世界）
- divine-human-conflict-flood（神人冲突洪水）

## 关联技法（怎么写）
- scene-description.md（场景描写）— 通过环境细节展示世界
- sensory-detail.md（感官细节）— 让读者"看到""听到"这个世界
- lightness-narrative.md（轻逸叙事）— 不要沉重地解释设定

## 关联风格（什么语感）
- 天渊风格：古典雅致的世界观描写
- 诸神愚戏风格：游戏化世界观展示

## 写作要点
1. "Show, don't tell"：通过角色行动展示规则，不要角色解释规则
2. 世界观信息分散在前10章，不要集中在第1章
3. 用角色的困惑/惊讶来引出世界观（"原来这个世界还有修仙者？"）
4. 世界观要有内在逻辑，不能随写随改
