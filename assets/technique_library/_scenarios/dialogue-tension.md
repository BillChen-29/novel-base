# 场景：对话张力（Dialogue Tension）

## 一句话定义
通过对话中的潜台词、信息博弈和情感冲突制造张力。

## 关联母题（对话冲突）
- brothers-betrayal（兄弟反目）— 对话中的背叛
- mentor-betrayal（师徒反目）— 对话中的师徒冲突
- false-accusation（诬陷）— 对话中的指控
- identity-mask（身份面具）— 对话中的伪装

## 关联角色（对话参与者）
- disguised-predator（伪装的掠食者）— 对话中隐藏意图
- hypocritical-white-moonlight（伪善白月光）— 对话中表演善良
- moral-grey-survivor（道德灰色求生者）— 对话中的道德博弈

## 关联技法（怎么写）
- dialogue-subtext.md（对话潜台词）— 表面说一件事，实际传达另一件事
- suspense-laddering.md（悬念阶梯）— 对话中层层递进的悬念

## 关联风格（什么语感）
- 末日降临风格：对话占比55-65%，对话驱动型叙事
- 天渊风格：对话占比45-55%，腹黑幽默对话

## 写作要点
1. 对话要有潜台词，不要直说
2. 每句话要有目的：展示性格/传递信息/推动剧情
3. 对话节奏：短句交锋，不要长篇大论
4. 对话中的沉默比语言更有力
