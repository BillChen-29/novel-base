# 场景：感情线（Romance）

## 一句话定义
角色之间的感情发展，需要暧昧、误会、表白等经典桥段。

## 关联母题（爱情类型）
- butterfly-lovers（梁祝）— 悲剧爱情
- cowherd-weaver（牛郎织女）— 跨越障碍的爱情
- fox-spirit-marriage（狐仙婚姻）— 超自然爱情
- snake-husband（蛇郎）— 变形者爱情
- divine-marriage（神圣婚姻）— 命运安排的爱情

## 关联角色（感情参与者）
- loyal-companion（忠犬伙伴）— 忠诚型恋人
- hypocritical-white-moonlight（伪善白月光）— 虚假爱情
- clever-wife（巧女）— 智慧型恋人
- mother-goddess（母神）— 守护型恋人

## 关联技法（怎么写）
- dialogue-subtext.md（对话潜台词）— 暧昧对话
- sensory-detail.md（感官细节）— 感官描写增强情感
- immersion-engineering.md（代入感）— 让读者感受角色的情感

## 写作要点
1. 感情发展要有节奏：相遇→暧昧→误会→和好→表白
2. 不要直接说"他爱她"，通过行为展示
3. 感情线要和主线交织，不要独立存在
4. 误会要合理，不要为了虐而虐
