# 场景：角色引入（Character Introduction）

## 一句话定义
新角色首次出场，需要快速建立辨识度、性格特征和叙事功能。

## 关联角色（出场方式参考）
- trash-turning-genius-mc（废柴逆袭主角）— 被欺辱后展示潜力
- cautious-immortal（苟道长生者）— 低调出场，隐藏实力
- loyal-companion（忠犬伙伴）— 主角危难时出手
- protective-elder（护犊子长辈）— 关键时刻庇护
- moral-grey-survivor（道德灰色求生者）— 灰色行为出场

## 关联母题（角色背景）
- abandoned-child-hero（被遗弃的英雄之子）
- reborn-avenger（重生复仇者）
- shapeshifter-archetype（变形者）

## 关联技法（怎么写）
- dialogue-subtext.md（对话潜台词）— 通过对话展示性格
- scene-description.md（场景描写）— 外貌+环境烘托
- immersion-engineering.md（代入感）— 让读者关注这个角色

## 写作要点
1. 首次出场必须有辨识度（外貌/动作/台词三选一）
2. 通过行为展示性格，不要直接说"他很聪明"
3. 角色功能要明确：是助力还是障碍？
4. 配角不要过度描写，留给读者想象空间
