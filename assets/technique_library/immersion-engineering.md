# 技法：代入感营造

> 让读者将自我投射到主角身上，实现"如果是我，我就会这样做"的沉浸体验

---

## 一句话定义
代入感营造是通过主角设计、视角选择和情节安排，让读者将自我意识置换到小说主角身上，从而获得"YY"（意淫）快感的网文核心技法。

## 原理
网络小说追求的不只是"好看"，而是让读者"成为"主角。传统大众文学虽然也取悦读者，但并不要求读者与主角合一——张无忌可以懦弱，郭靖可以迂腐，读者仍能接受。但网络小说的核心要求是：**主角的行为选择必须符合读者的期待**，让读者体验到"如果是我，我就会这样做"。

代入感的本质是"角色扮演"：读者摆脱各种现实义理的限制，在小说世界中"随心所欲"地生活。这与网络及ACG二次元文化带来的"沉浸体验"密切相关——网络让个体可以"代入"到某个更强大的理想自我身上，弥补现实人生的缺憾。

## 操作要点
1. **主角外貌要模糊化**：不要有确定的高矮胖瘦描写，"一人高"——留出空白让读者把自己想象成主角
2. **主角身份要贴近读者**：设定为小人物、普通人，预留升级空间；废物可以，但必须是"资质废"而非"废人"，要让读者看到逆袭希望
3. **主角性格要获得认可**：可以不完美，但必须有"不屈的精神"——不怕挫折、不折不挠。这是读者情感认同的基础
4. **故事环境要熟悉**：都市校园等现代读者熟悉的环境更容易代入；异世背景则通过"穿越"手法（现代人穿越过去）保持代入感
5. **尽早采用主角视角**：开篇就用主角视角，视角不能随意切换。配角不能抢主角戏份
6. **主角不能吃亏太久**：受辱后必须快速反弹，"做了任务不能没有收益"

## 适用场景
- 所有网络小说，尤其是玄幻、都市、穿越类
- 小说开篇的角色设定阶段
- 主角面临选择的关键情节
- 日常生活描写段落

## 经典案例
- 《盘龙》：林雷是普通少年→读者与他一起成长→从废柴到宇宙之主
- 穿越小说：主角是现代人→穿越到古代/异世→用现代思维解决问题→读者天然代入
- 起点白金写手"月关"：原本计划写古代土著主角，最终改为穿越者身份，理由就是"代入感"

## 常见错误
1. **主角太过完美**：完美主角让读者无法代入，因为"那不是我"
2. **主角性格让读者反感**：猥琐、虚伪、阴险会破坏代入感
3. **视角频繁切换**：读者刚代入主角就被拉到配角身上，打断沉浸
4. **虐主过度**：主角长期受辱不反弹，读者会弃文
5. **主角做出"傻"选择**：不符合读者期待的行为选择会引发愤怒
