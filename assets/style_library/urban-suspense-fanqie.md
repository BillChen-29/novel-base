# 番茄都市悬疑风格模板 (Urban Suspense - Fanqie Style)

## 1. 风格DNA

| 维度 | 设定 |
|------|------|
| **叙事视角** | 第三人称有限（贴近主角内心） |
| **时态** | 过去式 |
| **句式特征** | 短句为主（平均15-20字），长短交替制造节奏感 |
| **对话占比** | 40-50%（高对话驱动剧情） |
| **段落长度** | 3-5行（适配手机阅读屏） |

## 2. 语言风格

| 维度 | 设定 |
|------|------|
| **用词** | 口语与书面混合，避免文言词汇 |
| **修辞** | 低密度，偶尔一个精准比喻（每1000字不超过2个） |
| **感官描写** | 视觉+听觉为主，偶尔触觉增强紧张感 |
| **幽默类型** | 偶尔黑色幽默，不破坏紧张氛围 |

## 3. 番茄适配要点

### 开篇
- **前200字必须有冲突或悬念**，不要铺垫背景
- 直接把读者扔进事件中

### 章节结构
- **每章结尾必须有钩子**（悬念、反转、新信息）
- 单章2000-3000字为宜
- 每章至少一个小高潮

### 禁忌
- ❌ 避免长段心理描写（读者会跳过）
- ❌ 避免大段环境铺垫
- ❌ 避免缓慢的节奏

### 对话要求
- 对话要有**潜台词**，不说破
- 角色说话要有个人口癖/语气词
- 用对话推动剧情，不用对话说教

### 动作场景
- 用**短句加速**节奏
- 一句一个动作，不要复合句
- 适当留白制造紧张感

## 4. 反AI检查清单

- [ ] 避免"他感到一阵恐惧""她内心涌起一股暖流"等**情感标注**
- [ ] 避免过度总结（"这件事让他明白了一个道理……"）
- [ ] 避免成语堆砌（偶尔用一两个即可）
- [ ] 避免对称结构（"既……又……"、"不是……而是……"）
- [ ] 避免"仿佛""似乎""好像"高频出现
- [ ] 避免每段首句都是人名
- [ ] 避免过渡词滥用（"然而""不过""但是"连续出现）
- [ ] 对话不要每句都带"他说道""她回答"

## 5. 示范段落

### 开场段落示范（200字）

> 手机屏幕亮了一下。
>
> 林远睁开眼，凌晨三点十七分。一条陌生号码的短信：
>
> "她在我手里。你有两个小时。"
>
> 配了一张照片。模糊的背景，一只手绑着红绳。那是苏晚的手腕——他认得那条红绳，去年在寺庙求的。
>
> 他坐起来，后背全是汗。
>
> 窗外的城市还在沉睡，楼下的烧烤摊刚收完桌。一切正常得不像话。
>
> 手机又响了。第二条：
>
> "别报警。你可以试试，但后果自负。"
>
> 林远盯着屏幕，拇指悬在拨号键上方。报警，还是不报警？
>
> 苏晚到底在哪儿。

### 对话场景示范

> "你什么时候发现不对的？"赵队递了根烟过来。
>
> 林远没接。"昨天晚上。她没回消息，电话关机。"
>
> "以前呢？以前有没有过这种情况？"
>
> "没有。"林远停了一下，"……也不算没有。"
>
> 赵队手里的打火机停住了。"什么意思？"
>
> "上个月。她有三天没接我电话，说去外地出差。"林远的声音很平，像在说别人的事，"后来查了一下，那三天她根本没出城。"
>
> 赵队没说话。烟灰掉在桌上。
>
> "我问过她。她说工作上的事，不方便告诉我。"
>
> "你信了？"
>
> 林远看着窗外。"你觉得呢？"

### 动作场景示范

> 门开了。
>
> 林远闪身进去。走廊里没灯，手机光照出一面剥落的墙皮。
>
> 右手边。有声音。
>
> 他贴着墙往前挪。鞋底踩到什么东西，软的。他没低头看。
>
> 声音更近了。是呼吸声。一个人。或者两个。
>
> 手电筒照过去。
>
> 空的。
>
> 地上扔着一件衣服，深蓝色外套，还在动——不对，是风从破窗户灌进来，吹得衣摆晃。
>
> 他蹲下来摸了摸。还是温的。
>
> 刚走。

### 环境描写示范

> 城中村的巷子很窄，两边的房子几乎要碰到一起。头顶是乱拉的电线，像一张黑色的网。路灯只亮了一半，另一半坏了没人修。
>
> 水泥地上有水渍，散发出一股潮湿的霉味。林远低头看了一眼，水渍的形状像一只张开的手掌。
>
> 路边停着一辆面包车，车窗全是灰。后座上放着几个编织袋，鼓鼓囊囊的。
>
> 远处传来狗叫声，断断续续。
>
> 他加快了脚步。

## 6. 风格关键词速查

| 类型 | 关键词 |
|------|--------|
| **适合题材** | 悬疑、推理、都市犯罪、心理惊悚 |
| **节奏** | 快-慢-快交替 |
| **氛围** | 压抑、紧张、偶有温暖（用于反衬） |
| **适合平台** | 番茄小说、七猫等免费阅读平台 |
| **目标读者** | 18-35岁，手机碎片化阅读 |
| **参考风格** | 紫金陈、周浩晖、秦明（通俗化处理） |
