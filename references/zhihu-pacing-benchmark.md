## 二、段落规律

| 指标 | 知乎男频均值 |
|------|-------------|
| 段均长度 | 15-20字 |
| 最长段 | 40-75字 |
| ≤30字段占比 | 83% |

| 叙述段占比 | 68.7% |
| 短段(≤5字)占比 | 6.6% |

### 段落长度分布
```
1-5字    6.6%   短句/感叹/章节号
6-15字   38.5%  主力短段
16-30字  45.8%  主力长段
31-50字  8.7%   偶尔长段
51+字    0.4%   极少
```

### 段落节奏模式
- 叙述(15-25字) → 叙述(15-25字) → 对话(10-20字)
- 叙述(15-25字) → 短句(1-5字，情绪爆发)
- 数字标记 → 新场景开始
- 对话前必有叙述铺垫（75%对话段前有叙述）

## 四、知乎 vs 番茄核心差异

1. **篇幅**：知乎单篇 8-10k 完结 vs 番茄 100万字连载
2. **人称**：知乎几乎全第一人称 vs 番茄第三人称为主
3. **段落**：知乎段均 15-20 字 vs 番茄更长
4. **标点**：知乎更克制（问号 5.3 vs 番茄 14.1）
5. **钩子**：知乎按题材分化大（9.5-30.7）vs 番茄相对稳定（~33%）
