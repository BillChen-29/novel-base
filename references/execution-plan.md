## 关键 Pitfalls

1. **门禁 API 必须合并**——风格+AI+节奏+钩子合并一次调用，一致性本地完成，哲学对齐不走 API
2. **entity_chapter_map 归属**——子代理A生成维护，子代理C只读，无双写冲突
3. **原型重复检查**——仅本章引入新角色时触发
4. **自建方案备选**——探针失败时额外需 5-7 天
